How to Practice

1. Start with one octave: Play from the root note (red) to the next root note.
2. Use proper fingering: Typically one finger per fret (index=1, middle=2, ring=3, pinky=4).
3. Play ascending and descending: Go up the scale, then back down.
4. Use a metronome: Start slow and gradually increase tempo.
5. Learn the pattern: Major scale pattern is W-W-H-W-W-W-H (W=whole step, H=half step).
